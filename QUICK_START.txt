STEP 1: Install Dependencies (if not done)
================================================================================

Open PowerShell or Command Prompt in the project folder and run:

   pip install -r requirements.txt

If you have a virtual environment (evn folder), activate it first:
   .\evn\Scripts\Activate.ps1

Then install:
   pip install -r requirements.txt

================================================================================
STEP 2: Verify Setup
================================================================================

Run the verification script:
   python verify_setup.py

This will check if everything is configured correctly.

================================================================================
STEP 3: Setup Database
================================================================================

Run these commands:

   python manage.py makemigrations
   python manage.py migrate

This creates the database tables.

================================================================================
STEP 4: Create Admin User
================================================================================

Run:
   python manage.py createsuperuser

Enter:
   - Username: (choose any, e.g., "admin")
   - Email: (optional, press Enter to skip)
   - Password: (choose a password, remember it!)

================================================================================
STEP 5: Start the Server
================================================================================

Run:
   python manage.py runserver

You should see:
   "Starting development server at http://127.0.0.1:8000/"

Keep this window open!

================================================================================
STEP 6: Test the Application
================================================================================

1. Open your web browser
2. Go to: http://localhost:8000
3. Click "Login with Discord"
4. Authorize the app
5. You should see your dashboard!

================================================================================
IMPORTANT: Discord OAuth Setup
================================================================================

Before testing login, make sure in Discord Developer Portal:

1. Go to: https://discord.com/developers/applications
2. Select "GamingApp"
3. Click "OAuth2" in left sidebar
4. Under "Redirects", make sure this is added:
   http://localhost:8000/auth/discord/callback/
5. Save changes

If redirect URI is not added, login will fail!

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Module not found" errors
Solution: Install dependencies: pip install -r requirements.txt

Problem: "Discord OAuth not configured"
Solution: Check .env file exists and has correct values

Problem: "Invalid redirect URI" from Discord
Solution: Add redirect URI in Discord Developer Portal > OAuth2

Problem: Can't access admin
Solution: Make sure you created superuser: python manage.py createsuperuser

Problem: Port 8000 already in use
Solution: Use different port: python manage.py runserver 8001
